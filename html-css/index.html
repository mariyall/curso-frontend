Planejamento para criação de sites

Guia de criação de sites

Você chegou até aqui pois está pensando em fazer um novo site, se é a primeira vez, muitas dúvidas poderão surgir. Caso isso aconteça, você pode entrar em contato conosco.

Planejamento

Definição geral do protótipo do site.

É uma etapa bem abrangente pois é nela que deverão ser definidos todos os aspectos do projeto.

1. Quem vai acessar o site?

Qual o perfil do público

2. Qual o tipo de site mais apropriado?

Será institucional, um blog, um portal, uma loja virtual, entre outros.

3. Quais funcionalidades serão necessárias?

Seu site precisa de recursos para ser relevante para seus usuários?

4. Comece a produção!

Com as respostas das perguntas acima, certamente a ideia de onde se quer chegar estará mais clara.

Você pode começar:

  1. Desenhando a estrutura
  2. Escrevendo os textos de todas as páginas e seções do site.
  3. Pesquisar ou produzir fotos ou vídeos.

Se quiser pode compartilhar o resultado com a gente.
